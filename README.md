# travel-india

Hey everyone!

I would like to share that I have successfully created a website on June 28, 2023. This website is about all the amazing places you can visit in India. I worked hard on this project for three weeks. It is a responsive website, and I used programming languages like HTML, CSS, and JavaScript to make it. I also used tools like Visual Studio Code and GitHub.
 
On this website, you can find information and pictures of the best places to explore in India. From famous landmarks like the Taj Mahal to beautiful beaches and ancient cities,I tried to showcase India's wonders. I hope people enjoy using this website to learn more about these amazing places and may even plan their own trip to India!

Website Link: https://charan-021.github.io/travel-india/
